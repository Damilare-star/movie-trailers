## 🎮 Features to Test

1. **Homepage** - Browse trending, popular, and top-rated movies
2. **Hero Banner** - Click "Play Trailer" to watch movie trailers
3. **Search** - Use the search icon in navbar to find movies
4. **My List** - Add movies to your favorites list
5. **Genre Filter** - Browse movies by genre
6. **Movie Details** - Click any movie card to see full details
7. **Responsive Design** - Resize your browser to test mobile/tablet views

## 🛠️ Commands

- **Start Dev Server**: `npm run dev` (Already running!)
- **Build for Production**: `npm run build`
- **Preview Production Build**: `npm run preview`

## 🔧 If You Need to Restart

1. Stop the server: Press `Ctrl + C` in the terminal
2. Start again: `npm run dev`

## 📱 Test Responsive Design

- **Desktop**: Full width browser
- **Tablet**: Resize to ~768px width
- **Mobile**: Resize to ~375px width

## 🎨 Key Features to Explore

- Horizontal scrolling movie rows
- Smooth hover animations on cards
- Glassmorphism trailer modal
- Sticky navbar with blur effect
- Loading skeletons
- Scroll progress indicator
- Add/remove from favorites
- Search with debouncing
- Genre filtering

## 🐛 Troubleshooting

If you see any issues:
1. Make sure you're on `http://localhost:5174`
2. Check browser console for errors (F12)
3. Verify internet connection (needed for TMDB API)
4. Clear browser cache if needed
